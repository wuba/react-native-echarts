# 运行示例

1. 克隆 wrn-echarts 仓库

```shell
git clone git@github.com:wuba/react-native-echarts.git
```

2. 安装依赖

```shell
yarn
```

3. 切换到 example 目录

```shell
cd example
```

4. 运行 iOS 或 Android

```shell
yarn ios
# or
yarn android
```

这就是了! 你可以看到以下画面：
